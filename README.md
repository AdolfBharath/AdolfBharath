## Professional Summary

I build and operate defensive security capabilities that are measurable, automatable, and reliable in production.  
My focus is high-signal detection engineering, incident response workflows, and resilient telemetry pipelines.  
I integrate security controls into CI/CD with guardrails that scale across cloud and container platforms.  
I prioritize repeatability: policy-as-code, evidence-ready automation, and continuous improvement through feedback loops.
